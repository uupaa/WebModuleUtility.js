## How to use

### Browser

```js
<script src="lib/WebModuleUtility.js">
<script>
console.log( WebModuleUtility() );
</script>
```

### WebWorkers

```js
importScripts("lib/WebModuleUtility.js");

console.log( WebModuleUtility() );
```

### Node.js

```js
var WebModuleUtility = require("lib/WebModuleUtility.js");

console.log( WebModuleUtility() );
```
